# 🌿 EcoMarket

## Descripción del Proyecto

EcoMarket es una tienda en línea dedicada a la comercialización de productos ecológicos, reutilizables y sostenibles. El objetivo principal del proyecto es ofrecer una experiencia de navegación intuitiva y visualmente atractiva para usuarios interesados en el consumo responsable y el cuidado del medio ambiente.

Este proyecto fue desarrollado utilizando únicamente **HTML5 y CSS3**, siguiendo una filosofía **Mobile First** y respetando la restricción de no utilizar JavaScript ni frameworks externos.

---

## Objetivos

- Diseñar una interfaz moderna y accesible para una tienda ecológica.
- Aplicar buenas prácticas de maquetación semántica con HTML5.
- Implementar diseños responsivos mediante Flexbox y CSS Grid.
- Utilizar interactividad basada exclusivamente en CSS.
- Simular funcionalidades comunes de un e-commerce sin JavaScript.

---

## Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Pseudo-clases CSS (`:hover`, `:checked`, `:active`, `:valid`, `:invalid`)
- Selectores avanzados CSS

## Funcionalidades Implementadas

### Página de Inicio

- Header con navegación principal.
- Hero Banner con llamado a la acción.
- Sección de productos destacados.
- Efectos hover en tarjetas de productos.
- Footer informativo con enlaces y redes sociales.

### Catálogo de Productos

- Sidebar de filtros con posición sticky.
- Filtrado por categorías usando radios y CSS puro.
- Categorías:
  - Hogar
  - Moda
  - Cuidado Personal
  - Alimentos

- Filtrado por disponibilidad:
  - Todos
  - En Stock
  - Agotados

- Tarjetas de productos con:
  - Imagen
  - Nombre
  - Descripción
  - Precio
  - Estado de inventario
  - Sistema de favoritos mediante checkbox

- Diseño responsivo:
  - 4 columnas en escritorio
  - 2 columnas en tablet
  - 1 columna en dispositivos móviles

### Página Sobre Nosotros

- Historia de la empresa.
- Presentación de valores ecológicos.
- Principios de sostenibilidad.
- Testimonios de clientes.
- Animaciones CSS al interactuar con las tarjetas.

### Página de Contacto

- Formulario de contacto.
- Validaciones visuales utilizando:
  - `:valid`
  - `:invalid`

- Mapa ilustrativo.
- Redes sociales con animaciones hover.
